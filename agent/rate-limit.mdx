title: 'Защита от спама для чат бота GPT'
description: 'Защита от спама для чат бота GPT - лимит на количество сообщений'
---

Вы можете установить ограничение скорости на вкладке глобальных настроек вашего агента, чтобы ограничить количество сообщений, которые могут отправлять ваши пользователи.

**Этот параметр применяется только к следующей интеграции**

- Чат для сайта
- Чат в виде статичного окна
- Веб страница
