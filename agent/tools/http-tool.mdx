---
title: HTTP запросы к API
---

Инструмент HTTP — это очень мощная функция, позволяющая агентам ИИ беспрепятственно выполнять HTTP-запросы, позволяя вам вызывать любой API во время разговора. (Под капотом он использует вызов функций [https://platform.openai.com/docs/guides/function-calling](https://platform.openai.com/docs/guides/function-calling))

![Xnapper-2024-01-12-10.30.52.png](/images/http-tool/Xnapper-2024-01-12-10.30.52.png)

Этот инструмент интегрирован в языковую модель, которая использует как имя инструмента, так и его подробное описание, а также предопределенные параметры для определения подходящих моментов для активации.

Кроме того, модель может быть проинструктирована о том, когда использовать инструмент, с помощью директив, включенных в системную подсказку.

Например, примерное описание инструмента HTTP может быть таким:
**"Полезно для получения случайных изображений кошек."**

![Xnapper-2024-01-12-10.30.12.png](/images/http-tool/Xnapper-2024-01-12-10.30.12.png)

### Request Parameters

Редактор HTTP Tool облегчает создание необходимых переменных для выполнения HTTP-запроса.

Он включает опцию «Предоставлено пользователем», которая сигнализирует модели, что она должна собирать необходимые данные непосредственно из текущего разговора.

![Xnapper-2024-01-12-10.36.18.png](/images/http-tool/Xnapper-2024-01-12-10.36.18.png)

### **Разрешение**

При активации функции «Требуется разрешение» AI-агент будет ожидать авторизации от администратора, прежде чем приступить к обработке запроса.

Эта функция обеспечивает контролируемую и безопасную работу.
