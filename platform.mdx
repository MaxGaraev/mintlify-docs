Gptbots позволяют переносить данные из любого места (источники данных: текст, PDF, Word, PowerPpoint, Excel, Notion, Airtable, Google Sheets и т. д.) в хранилища данных (контейнер нескольких источников данных).

Затем ваши хранилища данных можно будет подключить к ChatGPT через плагины или любую другую модель большого языка (LLM) через API Gptbots.

**Gptbots создан для пользователей ChatGPT, разработчиков, предпринимателей и маркетологов.**

## Пояснения

| Название       | Описание                       |
| ---------- | --------------------------------- |
| Хранилище данных  | Контейнер с несколькими источниками данных |
| Источник данных | Формат загруженной информации                    |
| LLM        | Большая Языковая Модель              |
| Storage    | Векторная база данных        |
