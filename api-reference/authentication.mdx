Чтобы получить доступ к частному агенту или хранилищу данных, вы должны предоставить ключ API в заголовке авторизации вашего запроса.

```bash
'Authorization': 'Bearer <API_KEY>'
```

## API Keys

#### Вы можете получить ключ API на странице настроек своей учетной записи..

https://gptbots.chat/account
